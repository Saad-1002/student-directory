# Student Directory Portal

A React app showcasing a student directory with reusable StudentCard components, lists rendered with map(), and conditional badges. Built with Vite.

## Components
- **StudentCard**: Reusable component displaying name, rollNumber, course, email, skill badges, and active/inactive badge.

2. Run `npm install`
3. Run `npm run dev`
